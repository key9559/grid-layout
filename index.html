learn: grid layout

그리드 레이아웃

  • 그리드

    그리드 레이아웃 선언

    display: grid;
    display: inline-grid;
    컨테이너를 그리드로 만들어주는 속성.
    익스플로러 호환가능
    display: -ms-grid;

  • 높이/너비설정

    높이

    grid-template-row: 00px;
    grid-template-row: 1fr 3fr;
    items의 줄 높이 지정
    익스플로러 호환가능
    -ms-grid-rows: 1fr 3fr;
    -ms-grid-rows: 10px 40px;

    너비

    grid-template-columns: repeat(n,minmax(10% ayto));
    items의 줄 너비 지정
    익스플로러 호환가능
    -ms-grid-columns: 1fr 3fr;

    상단 두 속성 축약형

    grid-template: repeat(3,1fr)/ 2fr 2fr 2fr
    grid-template: "hd hd hd" 100px
    작성은 줄/칸
    아랫줄은 hd 3칸, 너비 100px

    이름으로 영역 생성

    grid-template-area: "hd hdside hd" "main main side" "ft . ft"
    아이템에 이름을 부여하여 제어하는 속성.
    .은 빈 공간이며, 각 이름을 item에 grid-area를 사용하여 부여해야 제어 가능

    오토 조정

    grid-auto-rows: 100px;
    grid-auto-rows: 100px 50px;
    grid-auto-columns: 100px;
    grid-auto-columns: 1fr 2fr;
    줄 혹은 칸 수를 미리 정할 수 없을 때 사용.
    수의 지정없이 높이 혹은 너비를 지정할 수 있음
    2개이상이 들어갈 땐 n번째로 반복된
    예제를 보면 2n은 50px, 2n-1은 50px의 높이를 가지고,
    2n의 너비는 1fr, 2n-1의 너비는 2fr이 된다

    전체 축약형

    grid: 1fr / 1fr;
    grid-auto-~, grid-template-~전체를 아우르는 축약형 속성

  • 간격: ie에선 마진으로 대체

    줄 사이 간격

    grid-row-gap: 10px;
    row-gap: 10px;
    줄과 줄 사이에 여백을 줄 때 사용

    칸 사이 간격

    grid-column-gap: 10px;
    column-gap: 20px;
    칸과 칸 사이에 여백을 줄 때 사용

    축약형

    grid-gap: 10px, 20px;
    gap: 10px, 20px;
    줄사이간격, 칸사이간격 순으로 작성
    줄끼리는 10px, 칸끼리는 20px 띄움

  • 배치

    아이템 배치흐름 지정

    grid-auto-flow: row
    아이템이 어떤순서로 배치되는지 지정
    속성값으로는
    row: 기본값. 가로 방향으로 순서대로 배치
    column: 세로방향으로 순서대로 배치
    row dense: 가로방향으로 순서와 상관없이 빈 곳을 채우며 배치
    dense: 상동
    column dense: 세로방향으로 순서와 상관없이 빈 곳을 채우며 배치
    가 있다.

  • 정렬

    아이템영역 수직정렬

    align-content: start;
    아이템 영역의 정렬을 지정
    속성값
    start: 시작점에
    center: 가운데에
    end: 끝에
    stretch: 컨테이너를 채우게 늘림
    space-between: 양끝에 배치하고, 가운데 컨텐츠들은 여백을 균일하게
    space-around: 아이템간의 여백을 균일하게
    space-evenly: 전체 여백을 균일하게 배치

  • 정렬

    아이템영역 수평정렬

    justify-content: start;
    아이템 영역의 정렬을 지정
    속성값
    start: 시작점에
    center: 가운데에
    end: 끝에
    stretch: 컨테이너를 채우게 늘림
    space-between: 양끝에 배치하고, 가운데 컨텐츠들은 여백을 균일하게
    space-around: 아이템간의 여백을 균일하게
    space-evenly: 전체 여백을 균일하게 배치

    축약형

    place-content: center space-between;
    place-content: center;
    두 정렬 방법이 같다면, 속성값은 1개가 온다.
    place-content: align-content justify-content;

  • 아이템내부 정렬

    세로정렬

    align-items: start;
    영역내 아이템의 세로를 정의함.
    속성값: stretch, start, center, end

    가로정렬

    justify-items: start;
    영역내 아이템의 가로를 정의함.
    속성값: stretch, start, center, end

    • 축약형

    place-items: start;
    place-items: center start;
    place-items: align-items justify-items;
    하나만 적으면 두 속성이 하나로 적용됨
